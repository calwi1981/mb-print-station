# MB Print Station

Android Bluetooth Druckstation fuer Web-POS pos.calwi.de.

Funktion:
- gekoppelte Bluetooth-Drucker anzeigen
- GOOJPRT 58mm Testdruck
- automatische Druckauftraege von https://pos.calwi.de/api/print_jobs_raw abrufen
- direkt ESC/POS ueber Bluetooth drucken
- Auftrag danach als gedruckt markieren

APK bauen:
GitHub Actions -> Build APK -> Run workflow
